# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating, you agree to uphold it: be respectful, assume good faith, and keep discussion constructive.

**Reporting:** if you experience or witness unacceptable behavior, report it privately through this repository's GitHub reporting tools. Maintainers will review and respond in good faith and will respect the privacy of reporters.
